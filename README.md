# GREAT EDID (macOS)
一款专为macOS设计的一键EDID注入工具，核心解决连接显示器时的显示异常问题，让外接屏幕稳定输出。

## 核心功能
- 修复HDMI连接下的**紫屏、绿屏、花屏**等显示异常
- 解决HDMI接口兼容性问题，实现稳定画质输出
- 注入EDID配置，无需复杂手动设置
- 支持Intel芯片与Apple Silicon（M系列）Mac
- 可选显示器图标自定义，不影响核心功能稳定性
- 一键备份与还原，操作安全无风险

## 适用场景
- 黑苹果的显示异常
- Mac通过HDMI连接外接显示器时出现色彩失真
- 显示器分辨率异常、画面闪烁或无法点亮
- 外接屏幕显示模糊、有条纹或色块
- macOS识别不到HDMI外接显示器或识别异常


## 使用步骤
1. 打开终端（Terminal），进入脚本所在文件夹
2. 运行脚本：将Great-EDID拖入终端中（需输入电脑密码，输入时无显示是正常现象）
3. 脚本自动识别所有连接的显示器，按提示选择需要修复的显示器序号
4. 选择 (1) 注入EDID
5. 提示“EDID注入成功，重启生效”后，重启Mac即可

## 推荐配置说明
强烈推荐「(6) 保持原样 + (1) 注入EDID」的组合，理由如下：
- 黑苹果可用此项
- 保持原有图标设置，避免系统识别冲突，兼容性更强
- EDID核心修复功能不受影响，专注解决HDMI显示问题
- 适合绝大多数显示器型号
- 大幅降低注入后出现二次问题的概率

3. 操作前建议备份重要文件，虽脚本自带备份机制，但避免极端情况

## 常见问题
- 注入后仍有显示异常？重新运行脚本选择“关闭EDID/重置设置”，重启后再次注入尝试
- 如何还原默认设置？运行脚本的功能（2）
- Apple Silicon（M系列）Mac无效果？确保系统已更新至最新版本，且显示器支持Mac兼容协议
